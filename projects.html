Redesigning Instagram's Privacy Settings

For my Usable Privacy and Security course, where my team worked to redesign Instagram’s privacy settings. We surveyed and interviewed participants, designed a new prototype based on their feedback and wrote a report about it. This required researching methods in the literature, combing through (with participants’ consent) social media profile, and writing a final report on resulting best practices.

A Game of Situational Awareness of Manatees and Boats

Last spring, I used Cocos Creator and JavaScript to create a video game about the dangers faced by manatees from boats. Cocos Creator had very limited documentation and it was often challenging to identify bugs. It was my first major programming project. I had never used Github before and had to learn the object-oriented programming concepts to improve the program. Through this process, I worked to optimize the code by studying the documentation that was available, using what I knew about programming, and thinking through JavaScript syntax.
